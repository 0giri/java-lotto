# 기능 요구사항

---

### 2단계 - 로또(자동)

- [x] 돈
    - 양의 정수만 값으로 가질 수 있다.
    - 가격이 주어지면 몇 개 구매 가능한지 조회
- [x] 로또 번호
    - 1 ~ 45 사이의 숫자만 가질 수 있다.
- [x] 로또
    - 6개의 로또 번호를 랜덤으로 가진다.
    - 같은 번호를 가질 수 없다.
- [x] 로또 생성기
    - 1000원 당 로또 1장을 발급해준다.
- [x] 로또 당첨 통계
    - 일치하는 로또 번호 개수 확인
    - 3개 이상 맞춘 로또 개수 확인
    - 수익률 계산

### 3단계 - 로또(2등)

- [x] 추가 당첨번호 입력 받기
- [x] 2등 당첨 방식 추가
    - 보너스 볼 포함 6개 매칭
- [ ] 2등 당첨 결과 출력 메세지 변경
